About this project

Engagement Arcade is a fast-paced, browser-based aim and reaction game built using HTML, CSS, and JavaScript. The game challenges players to quickly identify and click dynamically spawning targets within a limited time while maintaining streaks for bonus points and avoiding penalty targets. It features multiple difficulty levels that adjust spawn rates, target lifetimes, and risk, creating an engaging skill-based experience.

The project demonstrates core front-end concepts such as event-driven programming, real-time UI updates, animation timing, and complex game state management (start, pause, resume, and game over). A responsive HUD displays score, time, streak, and difficulty, while visual feedback and smooth animations reinforce player actions.

This game is inspired by Aim Lab, focusing on improving player precision, reaction time, and consistency through short, repeatable sessions. Engagement Arcade adapts these aim-training principles into a simplified, accessible mini-game designed to showcase interactive gameplay mechanics in the browser.
